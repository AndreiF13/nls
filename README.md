# nls [![npm-version][npm-badge]][npm-link]

List npm scripts.

![nls-screenshot][screenshot]

## Install & Usage

```bash
npm install -g nls
```

```bash
Usage

  $ nls [target-dir | -h | -v]

Examples

  # List npm scripts in current dir
  $ nls

  # List npm scripts in "node_modules/chalk"
  $ nls node_modules/chalk

  # Print this help
  $ nls -h
```
